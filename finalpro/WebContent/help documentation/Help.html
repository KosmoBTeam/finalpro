CSS Files and Structure - top

  • css/style.css - Most of the css styles can be found here. Each section is labeled for your convenience.
  • css/dark.css - This file contains most css for the dark styling.
  • css/bootstrap.css - This file contains the Bootstrap css.
  • css/responsive.css - This file contains the responsive settings.
  • include/rs-plugin/css/settings.css - This file contains css for the Revolution slider.

Change Accent Color - top

To change the yellow accent color simply replace all instances of the hexidecimal color #EAAF22 in css/style.css with one of your choosing. You can make the change in one swoop with most html editing programs such as Dreamweaver.

Change Skin Color - top

To change skin colors from light to dark simply add bgcolor-grey-dark to your class in the html which will set your background color to dark grey, and add dark which will change your text to a light color. For a simple exmaple of this see the file sup-page-dark.html, line 264.

There are several background colors preset for you to use:

  • bgcolor
  • bgcolor-grey
  • bgcolor-grey-light
  • bgcolor-grey-dark
  • bgcolor-black
  • bgcolor-red
  • bgcolor-green
  • bgcolor-blue

Navigation Skin Color - top

To change the Navigation bar from light to dark simply add the class dark to the navigation header. For a simple example of this see
sub-page-dark.html, line 43.

JavaScript - top

This theme imports several Javascript files.

  1. jQuery - jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. My Custom Scripts - Some of the animation in this site is carried out from the custom scripts.
  3. Plugins - In addition to the custom scripts, I implement a few "tried and true" plugins to create the effects. If you wish to customize any of the scripts you can find them in the folder "js"
  4. js/functions.js - Some modifications not available in the css can be made by editing this file.

Contact Forms - top

Simply add your email address to include/sendemail.php for the main contact form.

For the small contact widget add your email address to include/quickcontact.php

Subscribe Widget - top

In order for the subscribe widget to work you must set up an account with Mailchimp. It's free and very easy to do. Then simply enter your apiKey and listID inside include/subscribe.php.

Twitter - top

In order to make the twitter widget work you'll need to get the following from your twitter account:

  • API key
  • API secret
  • Access token
  • Access token secret

Enter the above inside include/twitter/tweets.php. Then add your twitter id to the html page code replacing the current id "envato". To see the twitter widget working you must view your site online, viewing locally won't work.

Disqus Comments - top

You can add a comments section just like a blog with Disqus comments. Simply register for free then add your comment shortname to the html code. You can find the disqus comments in the page "blog-single-small.html". To see the comments in action view online not locally. You can add a comments to any page not just a blog page.
